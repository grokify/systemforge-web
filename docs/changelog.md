## [Unreleased]

### Added

- `OrganizationMembersPage` component for managing organization members
  - Member list with avatars and role badges (owner/admin/member)
  - Edit member roles (admin/member)
  - Remove members from organization
  - Transfer ownership modal with confirmation (owner only)
- `OrganizationInvitationsPage` component for managing invitations
  - List pending invitations with expiration dates
  - Create new invitations (email + role selection)
  - Revoke pending invitations
  - View invitation history
- `OrganizationSettingsPage` component with tabbed navigation
  - General settings (organization name, logo URL)
  - Members tab placeholder
  - Invitations tab placeholder
  - Danger Zone (delete organization - owner only)
- New types for organization management:
  - `MemberRole` type (`'owner' | 'admin' | 'member'`)
  - `OrgSettingsTab` type (`'general' | 'members' | 'invitations' | 'danger'`)
  - `OrgMember` interface for member data
  - `OrgInvitation` interface for invitation data
  - Props interfaces for all organization management pages

### Documentation

- Updated `packages/pages.md` with organization management page documentation
- Added API reference for new page components and types
- Added permission model documentation
